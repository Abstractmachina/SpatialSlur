Detailed Description

Projection based constraint solver for geometry optimization and form-finding. This is an implementation of principles described in http://lgg.epfl.ch/publications/2012/shapeup/paper.pdf and follows many of the implementation details given in http://lgg.epfl.ch/publications/2015/ShapeOp/ShapeOp_DMSC15.pdf.
